## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

Esta aplicaion es una API de chat especiaalmente hecho para la aplicacion de cabal.
el chat es en tiempo real y se puede usar para enviar mensajes a todos los usuarios de la aplicacion.
el chat tambien da notificaciones a los usuarios de la aplicacion cuando alguien le envia un mensaje.
todas las conversaciones de este chat estan cifrado con un encriptado de seguridad end to end.

Esta aplicacion esta basada en el framework de nodejs , express y socket.io.
